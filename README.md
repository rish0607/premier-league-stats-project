# **🏆 Premier League Analytics Dashboard**

An interactive sports analytics dashboard built using Premier League season data. This project transforms raw football statistics into meaningful insights for analyzing team performance and player contributions.

---

## **📊 Features**

- **League Table**
    
    View team standings based on points, goal difference, and wins
    
- **Team Analysis**
    
    Explore metrics like win rate, goals scored, and goals conceded
    
- **Player Statistics**
    
    Access player data including goals, assists, appearances, and position
    
- **Advanced Metrics**
    - Expected Goals (xG)
    - Expected Assists (xAG)
    - Shot Creating Actions (SCA)
    - Progressive passes & carries
- **Performance Insights**
    - Goals per 90 minutes
    - Goal contribution per game
    - xG vs actual goals comparison

---

## **🛠️ Tech Stack**

- **Frontend:** React
- **Data Handling:** JavaScript / JSON
- **Visualization:** Chart.js / Recharts
- **Data Source:** FBref-style dataset

---

## **🎯 Objective**

To simulate real-world football analytics by converting complex datasets into interactive visual insights useful for performance analysis, scouting, and comparison.
